# 🔗 OrgLinks

Meu Organizador de Links — uma aplicação simples e prática para salvar, organizar e acessar seus links favoritos em um só lugar.

🔗 **Acesse online:** [org-links.vercel.app](https://org-links.vercel.app)

## ✨ Sobre o projeto

O OrgLinks foi criado para facilitar a organização de links importantes do dia a dia, evitando bagunça de abas salvas ou anotações espalhadas.

> **Observação:** este é um projeto pessoal, desenvolvido como parte dos meus estudos em desenvolvimento web. Ele foi personalizado para atender às minhas necessidades específicas de uso, então algumas escolhas de design e funcionalidade refletem preferências pessoais em vez de um caso de uso genérico.

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## 📦 Como rodar localmente

1. Clone o repositório:
   ```bash
   git clone https://github.com/dionesart/OrgLinks.git
   ```
2. Entre na pasta do projeto:
   ```bash
   cd OrgLinks
   ```
3. Abra o arquivo `index.html` no navegador (ou use a extensão Live Server, se estiver usando o VS Code).

## 📁 Estrutura do projeto

```
OrgLinks/
├── assets/       # Imagens e recursos estáticos
├── index.html    # Página principal
├── script.js     # Lógica da aplicação
└── style.css     # Estilização
```
